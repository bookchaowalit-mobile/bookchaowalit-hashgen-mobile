# Hashgen — Mobile

Flutter mobile app for **Hashgen**.

Part of [Chaowalit Greepoke](https://bookchaowalit.com)'s 101 Portfolio Projects.

## Tech Stack

- **Framework:** Flutter 3.24 + Material 3
- **Language:** Dart
- **State:** Riverpod
- **Navigation:** GoRouter

## Getting Started

```bash
flutter pub get
flutter run
```

## Build

```bash
# Android
flutter build apk --release

# iOS
flutter build ios --release
```
